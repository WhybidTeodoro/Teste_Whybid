# Teste de Integração com API Pública - ANS

Este projeto faz parte do processo seletivo para estágio na Intuitive Care.

O objetivo é integrar com a API pública de dados abertos da ANS, processar arquivos
de demonstrações contábeis dos últimos 3 trimestres e consolidar informações de
despesas com eventos/sinistros em um único arquivo CSV.

## Tecnologias
- Python 3
- Git/GitHub

## Estrutura do Projeto
- `src/`: código fonte
- `data/`: dados brutos, extraídos e processados
